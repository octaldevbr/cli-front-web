# Template - Front Web

Para iniciar o projeto, execute:

```bash
npm run start
```

Para fazer uma build de produção:

```bash
npm run build
```

Para fazer levantar um servidor com os arquivos de produção:

```bash
npm run start:build
```
------------------------
